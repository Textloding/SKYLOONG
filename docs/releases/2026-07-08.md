# 2026-07-08 版本说明

本版本是 SKYLOONG/GK87 ESP32-S3 屏幕固件的现代化体验版，主分支面向普通用户和二次开发者使用。

## 重点变化

- 重做网页管理后台，支持桌面端和手机端自适应访问。
- 增强长期运行后的网页服务稳定性，增加 `/health` 诊断接口。
- 支持保存多个 Wi-Fi，并在开机时自动尝试重连。
- 恢复 QR/Wi-Fi 配网页进入普通 `fn + ~` 切换链，确保首次配网入口可达。
- 优化 GIF/MP4 上传转码，默认面向 `320 x 240` 屏幕输出。
- 增加视频声音开关，用户可在流畅度和声音之间取舍。
- 增加番茄钟独立页面，支持主题、提醒音、当前倒计时重置和轮次重置。
- 增加键盘宠物页面，支持开关、主题、互动和汪汪音效开关。
- 增加内置轻量按键音效，减少对外部音频文件的依赖。
- 天气城市支持自动识别和手动设置，管理台不再直接展示天气 API Key。

## 刷机建议

普通用户建议使用独立刷机工具：

```text
https://github.com/Textloding/skyloong-flasher
```

开发者仍可使用 ESP-IDF v5.1.4 手动编译和刷机：

```powershell
idf.py build
idf.py -p COM3 flash
```

## 注意事项

- 屏幕物理分辨率为 `320 x 240`，高分辨率视频会被压缩、裁切或补边。
- ESP32-S3 不是高清视频播放器，视频流畅度优先时建议关闭视频声音并重新上传。
- 设备只支持 2.4GHz Wi-Fi。
- 刷机后首次联网、启动网页服务可能需要 10 到 60 秒，请等待屏幕显示 IP 后再访问管理台。
- 如果只看到 `COM1`，通常不是屏幕刷机串口；请寻找包含 `303A:1001` 或 Espressif 信息的串口。
